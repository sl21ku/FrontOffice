# FrontOffice

> 試合は見ない。勝敗は、冬に決まる。

架空プロ野球12球団の球団フロントとして、オフシーズンの補強と放出だけでペナントを動かす編成シミュレータ。

## できること

- **球団選択** — 12球団から予算・戦力傾向を見て所属球団を選ぶ
- **選手一覧** — 野手/投手切替、打率・本塁打・防御率・勝利など個別ソート可能
- **戦力外通告** — 不要選手を放出し年俸枠を確保
- **FA交渉** — 金額と年数を提示してFA選手を獲得。自チームからのFA宣言選手には残留交渉も可能
- **人的補償** — A/BランクFA獲得時にプロテクトリスト編成
- **トレード** — 他球団と選手交換（成立可能性をリアルタイム表示）
- **ドラフト** — 全球団1巡目同時指名＋競合抽選のNPB方式。スカウト評価・球種・最速・アマチュア成績つき
- **現役ドラフト** — 出場機会の少ない選手を対象にしたNPBルール準拠の入れ替え
- **ポスティング** — 海外移籍希望選手の容認/否認
- **ペナントシミュレーション** — 143試合を一括計算し順位・チーム成績・個人成績を表示
- **全球団記録** — 全選手のデータを一覧・ソート可能
- **年俸調整** — 適正価格を参考に選手年俸を手動設定。割安だとFA流出リスク増＋能力低下

## 遊び方

1. `index.html` をブラウザで開く
2. 球団を選択
3. 戦力外・FA・トレード・ドラフト・現役ドラフトで編成
4. ペナント開幕 → 結果確認
5. 翌年へ進む → 繰り返し

## 技術

HTML + CSS + JavaScript（vanilla）。ブラウザのローカルストレージにセーブ。
