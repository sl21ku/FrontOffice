# FrontOffice

> 試合は見ない。勝敗は、冬に決まる。

架空プロ野球12球団の球団フロントとして、オフシーズンの補強と放出だけでペナントを動かす編成シミュレータ。
NPBの制度を参考にした架空ルールを採用。実在球団・実在選手は一切使用していない。

## 実装済み機能

- **球団選択** — 12球団から予算・戦力傾向・リーグ環境を見て所属球団を選ぶ
- **選手一覧** — 野手/投手切替、打率・本塁打・打点・防御率・勝利・奪三振など個別ソート可能
- **戦力外通告** — 不要選手を放出し年俸枠を確保
- **FA交渉（金額・年数提示）** — 自チームFA宣言選手の残留交渉、他球団FA選手の獲得交渉
- **人的補償** — A/BランクFA獲得時にプロテクトリスト編成
- **トレード** — 他球団と選手交換（成立可能性をリアルタイム表示）
- **ドラフト** — 1巡目全球団同時指名＋競合抽選＋外れ再指名、2巡目以降ウェーバー順
- **現役ドラフト** — 出場機会の少ない選手を対象にした入れ替え
- **ポスティング** — 海外移籍希望選手の容認/否認
- **ペナントシミュレーション** — 143試合一括計算、順位・チーム成績・個人成績
- **全球団記録** — 全選手データの一覧・ソート
- **年俸調整** — 適正価格を参考に手動設定（割安だとFA流出リスク増＋能力低下）
- **リーグ環境設定** — 投高打低 / 標準 / 打高投低
- **クリック音**

## 遊び方

1. `index.html` をブラウザで開く
2. リーグ環境を選び球団を選択
3. 戦力外・FA・トレード・ドラフト・現役ドラフトで編成
4. ペナント開幕 → 結果確認
5. 翌年へ進む → 繰り返し

## 技術

HTML + CSS + JavaScript（vanilla）。ブラウザのローカルストレージにセーブ。
